# 小黑书 Xiaoheishu

> 意义经济基础设施 — 帮助每个人把独特的生活体验变为有价值的内容，并在发布、传播、变现的完整闭环中获益。

---

## 一句话定位

**小黑书是一个去中心化的内容创作与分发平台**：用户通过 AI 眼镜/手机捕捉生活，本地 AI 辅助创作，一键发布到多平台，并通过 Spores 协议实现内容→流量→收入的无许可分成。

---

## 意义经济（Meaning Economy）

随着 AI 快速替代社会化再生产中的劳动力角色，人类不可替代的价值在于**独特的生活体验**。

小黑书认为人在意义经济中扮演三种角色：

| 角色 | 定义 | 典型行为 |
|------|------|---------|
| **表达者** | 分享自己独特的生活/旅行/美食体验 | 拍照、写感受、发布 |
| **创造者** | 用知识和想象力探索问题、给出方案 | 写分析、做研究、提出新可能 |
| **建设者** | 看到好想法后把它变为现实 | 开发工具、搭建基础设施、组织社区 |

任何人都可能在这三个角色间切换。区别于传统平台"平台获益、用户劳动"的模式，小黑书让**每个角色都能从自己的贡献中获得归属于个人的价值回报**。

## Spores 协议（包子协议）

**核心逻辑**：表达者分享了真实体验 → 为商家带来新客户 → 商家通过智能合约自动分成给表达者

```
表达者发布体验内容
       ↓
新客户通过内容找到商家（带唯一 barcode）
       ↓
商家获得可验证的链上归因订单
       ↓
智能合约自动分成 → 表达者获得奖励
```

- **无许可加入**：商家和表达者均可自由加入，无需平台审批
- **跨链声誉**：reputation 归属于个人账号（基于 AirAccount），永久存在
- **打赏机制**：用户可对喜欢的内容直接打赏
- **City 协议**：每个城市 3-5 个社区节点提供基础服务，分散化运营

底层基础设施：AAStar AirAccount、SuperPaymaster、CometENS、COS72、MyNFT、MyPNTs、MyShops

## 产品矩阵

### 1. AI 眼镜（理想输入终端）
- 按两下自动开始录制照片/视频/声音
- 手机不在手边也能捕捉精彩瞬间（手机在包里/口袋里时不会错过）
- 通过蓝牙/WiFi 同步素材到手机 App

### 2. 手机 App（主创作终端）
- 接收眼镜素材，或直接拍照/录视频
- **本地 ASR/TTS**：摁住说话替代打字（本地小模型如 Whisper，数据不出设备）
- 说一句"我刚吃了家很好吃的海鲜，价格超便宜，两个人100块"→ App 自动生成结构化草稿
- 一键发布到小黑书，可选同步到其他平台
- 连接眼镜后素材自动导入，适度编辑后快速发布

### 3. 桌面 App（多平台发布终端）
- **本地内容管理**：所有内容存储在本地（SQLite + Markdown），内嵌 Web Server，本地预览如网站
- **多平台自动发布**：小红书、微信公众号、Twitter/X、知乎、Instagram 等
- 不想发布到网上？纯本地留存也支持
- 内容可打包备份到 iCloud / 本地磁盘
- 技术路线：Electron + CDP（复用用户现有浏览器 session，规避平台反自动化检测）

### 4. 后台自动服务（用户无感知）
- 内容自动审核与隐私过滤
- SEO/GEO 自动优化（结构化数据注入、hreflang、sitemap 生成）
- 图片自动压缩与优化
- 多语言自动翻译（zh ↔ en）
- 内容质量 AI 润色（可选）

## AI 模型使用

| 场景 | 模型类型 | 运行位置 |
|------|---------|---------|
| 语音输入（ASR）| 本地小模型（Whisper 系列）| 设备端 |

| 内容优化/润色 | 大模型 API | 云端（可选） |
| 图片处理/生成 | 本地或云端 | 混合 |
| 隐私过滤 | 本地小模型 | 设备端 |
| GEO 结构化 | 规则 + 大模型 | 云端 |

## 与中心化平台的本质区别

| 维度 | 小红书 / Facebook / 微博 | 小黑书 |
|------|------------------------|-------|
| **内容归属** | 平台条款默认取得内容授权，UGC 成为平台资产 | 内容永远归用户所有，平台无权占有 |
| **搜索引擎** | 屏蔽爬虫，内容被困在平台围墙内 | 搜索引擎友好，内容可被 Google / AI 引擎索引 |
| **广告模式** | 在你的内容旁插入广告，收益归平台 | **零广告**，用 Spores 协议替代——自愿加入，随时开关 |
| **变现方式** | 平台决定分成比例，随时可以改规则 | 智能合约执行，规则透明，无需平台许可 |
| **账号风险** | 平台可随时封号、删内容 | 内容存储在用户本地 + 去中心化网络，平台无法删除 |

> **核心立场**：你的体验是你的资产，不是平台的免费内容库。
